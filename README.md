# Game-hub-Project
Project: Python Games Hub

A user authentication system is developed along with three engaging games : 
1. Quiz Game: A trivia game that challenges players to answer questions correctly from four options.
2. Number Guessing Game: A game of chance where players attempt to guess a random number within given attempts.
3. Tic Tac Toe: A classic game of strategy and skill.
4. User Authentication system: Data is stored  even after closing of program and can login whenever want with correct id and password.

Features:

- User authentication system for secure login and gameplay
- Three unique games with varying levels of difficulty
- Python programming language used for development
